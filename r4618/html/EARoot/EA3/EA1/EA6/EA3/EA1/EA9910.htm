-- Name --<br /></p><p>GridObservation<br /></p><p><br /></p><p>-- Definition --<br /></p><p>Observation representing a gridded field at a single time instant.<br /></p><p>-- Description --<br /></p><p>A GridObservation is an observation of some phenomenon (or phenomena) over a gridded field. E.g. Output from a model, or rectified, georeferenced satellite data.<br /></p><p>The result of a GridObservation is a discrete coverage within a compound spatiotemporal CRS where the domain consists of a two- or three-dimensional grid of points, all having the same time instant temporal component.<br /></p>
