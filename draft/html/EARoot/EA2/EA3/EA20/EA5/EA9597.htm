-- Name --<br /></p><p>Statistical grid cell<br /></p><p><br /></p><p>-- Definition --<br /></p><p>Unit for dissemination or use of statistical information that is represented as a grid cell.<br /></p><p><br /></p><p>-- Description --<br /></p><p>A statistical grid cell is associated with:<br /></p><p>- the unique statistical grid cell of its upper level (if any) it is covered by,<br /></p><p>- the statistical grid cells of its lower level (if any) it covers.<br /></p><p><br /></p><p>NB Statistical grid cells are squares.<br /></p><p><br /></p><p>EXAMPLE In a quadtree structure, some cells are associated with the four cells they are decomposed into.<br /></p><p><br /></p>

Notes: -- Name --<br /></p><p>Code<br /></p><p><br /></p><p>-- Definition --<br /></p><p>A cell code.<br /></p><p><br /></p><p>-- Description --<br /></p><p>This code is composed of:<br /></p><p>1. A coordinate reference system part, represented by the word CRS, followed by the EPSG code.<br /></p><p>2. A resolution and position part:<br /></p><p>- If the coordinate reference system is projected, the word RES followed by the grid resolution in meters and the letter m. Then, the letter N followed by the northing value in meters, and the letter E followed by the easting value in meters too.<br /></p><p>- If the coordinate reference system is not projected, the word RES followed by the grid resolution in degree-minute-second, followed by the word dms. Then the word LON followed by the longitude value in degree-minute-second, and word LAT followed by the latitude value in degree-minute-second.<br /></p><p>For both cases, the given position is the position of the lower left cell corner.<br /></p><p><br /></p><p>NOTE: This code acts like an identifier.<br /></p><p><br /></p>
