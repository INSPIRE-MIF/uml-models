-- Definition --<br /></p><p>The generic habitat class.  <br /></p><p><br /></p><p>-- Description --<br /></p><p>The classification is used by Natura2000, item 4.1 of the Standard Data Form.  This classification is similar to Habitat Type, but is more general.  However, there is no simple mapping from Habitat Class to Habitat Type, and Natura 2000 requires both, so it is retained.  Ultimately, however, this duplication should be addressed.<br /></p>
