-- Definition --<br /></p><p>Extended concrete types common to overall Utility Networks sub-domain. <br /></p><p><br /></p><p>-- Description --<br /></p><p>Each of the common concrete elements may be used as part of one or multiple networks. E.g. manhole type is a container, and may contain features belonging to both water and sewer networks. Furthermore, the pole feature (also a container) may hold both telecommunications features and features belonging to electricity networks.<br /></p>
