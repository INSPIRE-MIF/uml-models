-- Name --<br /></p><p>CPA code value<br /></p><p><br /></p><p>-- Definition --<br /></p><p>The CPACodeValue code list hosts the family of reference values for the "product" item referred to the attribute processItem in the ProcessInput or ProcessOutput classes.<br /></p><p>A product means something that is produced, result of manufacturing, a result of an action or a process.<br /></p><p><br /></p><p>-- Description --<br /></p><p>CPA is the Standard Classification of Economic Products from Annex Regulation (EC) n. 451/2008. It has a hierarchical structure funded on different levels embedded in the activity code referring to the product. The activity code is accompanied by  the activity denomination.<br /></p>
