-- Name --<br /></p><p>derived soil profile<br /></p><p><br /></p><p>-- Definition --<br /></p><p>non-point-located soil profile that serves as a reference profile for a specific soil type in a certain geographical area.<br /></p><p><br /></p><p>-- Description --<br /></p><p>The characteristics of a derived soil profile are mostly derived (e.g. averaged) from one or several observed profiles of the same soil type in the area of interest, or are designed with expert knowledge about the same kind of landscape.<br /></p><p><br /></p><p>NOTE 1 The derived soil profile represents the average or typical profile that characterizes the so called soil typological unit, soil series.<br /></p>
