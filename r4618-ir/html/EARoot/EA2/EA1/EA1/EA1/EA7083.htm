-- Definition --<br /></p><p>Information defining the specification used to create or derive this geographic position of the address.<br /></p><p><br /></p><p>-- Description --<br /></p><p>NOTE 1 Multiple address points can be derived from one polygon spatial object. <br /></p><p><br /></p><p>NOTE 2 If the position of an address is derived from a polygon spatial object a number of different approaches is used.<br /></p><p><br /></p><p>EXAMPLE 1 The same point (e.g., centre point of the polygon) is used for each address, thus, multiple address points will be overlapping.<br /></p><p><br /></p><p>EXAMPLE 2 Each point position is unique within the polygon to be able to visually distinguish the representation of each address. <br /></p>
