-- Name --<br/>soil theme descriptive coverage<br/><br/>-- Definition --<br/>a spatial object type that is associated to the soil theme coverage and holds additional information on values of a property of the soil theme coverage.<br/><br/>-- Description --<br/>This coverage can only exist in relation to a base coverage SoilThemeCoverage, it has the same domain and spatial extent as the base coverage. And it gives extra information on the values of the base coverages<br/>SOURCE Adapted from "Coverage" [ISO 19123:2005].<br/>EXAMPLE coverage indicating the confidence level of the pH values reported in the associated SoilThemeCoverage.<br/>
