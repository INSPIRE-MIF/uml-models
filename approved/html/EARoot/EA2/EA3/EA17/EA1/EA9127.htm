-- Name --<br/>soil plot<br/><br/>-- Definition --<br/>spot where a specific soil investigation is carried out.<br/><br/>-- Description --<br/>NOTE 1: For observed soil profiles, a spot location has to be defined. It is the place on which the observation is made. The profile is an observation made on the plot.<br/><br/>NOTE 2: All types of plots only provide locality, but no soil information itself. <br/><br/>EXAMPLE A borehole is the location where you gather the information to abstract a profile information from.<br/><br/>SOURCE GS Soil<br/><br/>
