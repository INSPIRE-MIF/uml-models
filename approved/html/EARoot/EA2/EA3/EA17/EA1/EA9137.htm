-- Name --<br/>soil derived object<br/><br/>-- Definition --<br/>A spatial object type for representing spatial objects with soil-related property derived from one or more soil and possibly other non soil properties.<br/><br/>-- Description --<br/>NOTE Soil thematic maps can be derived directly from the involved soil database (organic matter content, pH, texture, etc.) or they can be derived by using pedotransfer functions or pedotransfer rules (e.g. plant available water in the rooting depth). Derivation can be simple extraction from a single data field, or a complex combination of different kind of data and application of e.g. mathematical or expert knowledge-based procedures.<br/>
