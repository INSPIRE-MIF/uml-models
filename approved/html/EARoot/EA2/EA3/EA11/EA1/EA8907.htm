«featureType» MiningFeatureOccurrence
Class   		Name: specification
Indicates the MiningFeature that the occurrence specifies Points to any number of MiningFeatureOccurrences which are occurrences of the Miningfeature. MiningFeatureOccurrences may all be from a single map or from several maps, or reflect different levels of detail, eg point, area, 3D volume. 		Name: occurrence
Point to the single MiningFeature which describes the Mining feature Occurrence
Details:
a description association that links a notional mining feature with any number of occurrence features.  A mining feature, such as a Mine may be linked to mining feature occurrences from a number of different maps or as a point, area or 3D mine plans.  A mining feature occurrence, however is always associated with only a single description (MiningFeature).<br/><br/>
