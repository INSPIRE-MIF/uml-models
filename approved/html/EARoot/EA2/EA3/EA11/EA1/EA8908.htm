-- Definition --<br/>A spatial representation of a MiningFeature.<br/><br/>-- Description --<br/>A MiningFeatureOccurrence provides a link between a notional feature (description package) and one spatial representation of it, or part of it. The MiningFeatureOccurrence carries a geometry and the association with a Mining Feature provides specification of all the other descriptors.<br/>          <br/><br/><br/>

Details:
a description association that links a notional mining feature with any number of occurrence features.  A mining feature, such as a Mine may be linked to mining feature occurrences from a number of different maps or as a point, area or 3D mine plans.  A mining feature occurrence, however is always associated with only a single description (MiningFeature).<br/><br/>
