-- Definition --<br/>Concrete types common to overall Utility Networks sub-domain. <br/><br/>-- Description --<br/>Each of the common concrete elements may be used as part of one or multiple networks. E.g. manhole type is a container, and may contain features belonging to both water and sewer networks. Furthermore, the pole feature (also a container) may hold both telecommunications features and features belonging to electricity networks.<br/><br/>
