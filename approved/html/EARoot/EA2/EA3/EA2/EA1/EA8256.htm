-- Name --<br/>Building nature value<br/><br/>-- Definition --<br/>Values indicating the nature of a building.<br/><br/>-- Description --<br/>NOTE 1 : This code list does not aim to be exhaustive as the attribute buildingNature addresses only noticeable buildings.<br/>NOTE 2: The values included in this code list address mainly (but not only) two international use cases: air flights where buildings may be obstacles and marine navigation where buildings may be landmarks.<br/>NOTE 3: This code list should only be applied for buildings, even if it may be applicable to other constructions (for example, not all dams are buildings).<br/><br/>
