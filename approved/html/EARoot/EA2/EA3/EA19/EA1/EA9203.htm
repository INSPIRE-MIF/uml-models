-- Name --<br/>Statistical unit<br/><br/>-- Definition --<br/>Unit for dissemination or use of statistical information.<br/><br/>-- Description --<br/>SOURCE [INSPIRE Directive:2007].<br/><br/>EXAMPLE grid cell, point, line, polygon.<br/><br/>NOTE Spatial features of any INSPIRE application schema can be considered as a statistical unit, because all can be used as spatial reference.
