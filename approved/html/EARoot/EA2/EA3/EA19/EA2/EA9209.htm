-- Name --<br/>Statistical grid cell<br/><br/>-- Definition --<br/>Unit for dissemination or use of statistical information that is represented as a grid cell.<br/><br/>-- Description --<br/>A statistical grid cell is associated with:<br/>- the unique statistical grid cell of its upper level (if any) it is covered by,<br/>- the statistical grid cells of its lower level (if any) it covers.<br/><br/>NB Statistical grid cells are squares.<br/><br/>EXAMPLE In a quadtree structure, some cells are associated with the four cells they are decomposed into.<br/><br/><br/>

Notes: -- Name --<br/>Code<br/><br/>-- Definition --<br/>A cell code.<br/><br/>-- Description --<br/>This code is composed of:<br/>1. A coordinate reference system part, represented by the word CRS, followed by the EPSG code.<br/>2. A resolution and position part:<br/>- If the coordinate reference system is projected, the word RES followed by the grid resolution in meters and the letter m. Then, the letter N followed by the northing value in meters, and the letter E followed by the easting value in meters too.<br/>- If the coordinate reference system is not projected, the word RES followed by the grid resolution in degree-minute-second, followed by the word dms. Then the word LON followed by the longitude value in degree-minute-second, and word LAT followed by the latitude value in degree-minute-second.<br/>For both cases, the given position is the position of the lower left cell corner.<br/><br/>NOTE: This code acts like an identifier.<br/><br/><br/>
