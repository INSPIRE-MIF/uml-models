-- Name --<br/>Marine Contour<br/><br/>-- Definition --<br/>A set of isolines representing the value of some phenomenon at a particular time.<br/>-- Description --<br/>A representation of any physical or chemical phenomenon related to the SeaArea.  MarineContour should be derived from an OceanGeographicFeature of the same phenomenon.  The source observations used to derive this feature may be linked to via the sourceObservations association. MarineContour is an abstraction of the ‘real world’ to provide an overview of the expected conditions of a SeaArea.  Examples would include salinity contours, depth contours, significant wave height.<br/>
