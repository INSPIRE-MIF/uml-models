-- Definition --<br/>A multi-modal itinerary is calculated based on starting point, destination and travel details provided by the user through a web interface. Travel details include time of departure/arrival and mode of transport. A proposed itinerary is displayed on the screen and can be printed by the user.<br/>

User types in a location for the starting point or searches in a list of known locations.<br/><br/>User types in a location for the destination or searches in a list of known locations<br/><br/>User chooses time of departure or arrival.<br/><br/>User selects one or more acceptable modes of transport.<br/><br/>The system checks starting point and destination locations (address or transport network node) against a database of known locations.<br/>If both locations are found, the system calculates an itinerary.<br/><br/>If an itinerary is found, it is displayed on the screen.<br/><br/>User prints out details of itinerary.<br/>
