-- Definition --<br/>A collection of transport link sequences and or individual transport links that has a specific function or significance in a transport network. <br/><br/>-- Description --<br/>NOTE<br/>This spatial object type supports the aggregation of links to form objects with branches, loops, parallel sequences of links, gaps, etc.<br/><br/>EXAMPLE<br/>A dual carriageway road, as a collection of the two link sequences that represent each carriageway. <br/><br/>
