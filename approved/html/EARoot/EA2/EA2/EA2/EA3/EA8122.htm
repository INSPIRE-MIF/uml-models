-- Definition --<br/>Generic class for collections of geophysical objects<br/><br/>-- Description --<br/>It is a set of geophysical objects that are grouped by some common property. p.e: created in the same measuring campaign. GeophObjectSets are used for spatial sampling either by means of data acquisition or data processing. The produced  result of a geophObjectSet is always collective, e.g. a map constructed from the results of the individual member objects.<br/><br/><br/>
