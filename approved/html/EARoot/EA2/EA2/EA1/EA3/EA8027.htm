-- Name --<br/>void area<br/><br/>-- Definition --<br/>Area of the Earth's surface where the elevation model is unknown because of missing input data. This area shall be excluded from a DEM.<br/><br/>-- Description --<br/>SOURCE "ObscureArea" [GIS4EU D3.5].<br/><br/>NOTE 1 A void area may contain a number of isolated areas with elevation data. These are considered as holes of the container void area.<br/><br/>NOTE 2  Void areas can be useful as masks when calculating a more appropriate or accurate DEM.<br/><br/>
