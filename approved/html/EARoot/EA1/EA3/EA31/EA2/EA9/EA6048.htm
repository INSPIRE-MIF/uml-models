This example is a translation of a class from the CORINNE LCCS. The CORINNE description is "annual crops or grazing land under the wooded cover of forest species". <br/>The model illustrated in Figure C9 shows that two layers are used.  The first one composed by either the LCML element Herbaceous Vegetation (with an extra characteristic Cultivated) or the LCML element Herbaceous Vegetation (with an extra characteristic Grazed). The second one composed by the LCML element Tree. The cover of the trees is not indicated in the CORINNE class description; however it has been interpreted in the translation as a rather open layer (cover between 5 to 30 %). In LCML the user can avoid to specify the property Cover. In this translation the property Cover has been inserted to better characterize the functional relationship between the agricultural layer and the wooded one.<br/>
