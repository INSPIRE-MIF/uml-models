The type "MM_TransferNode" is a topological node which has co-boundaries with potential route segments on the multimodal network. At a transfer node, a traveling mode changes or a transfer occurs. A transfer node is either a fixed point (e.g. bus stop, subway station, etc.) or an arbitrary point (e.g. any point on a road to take a taxi) on the base network. <br/>
