-- Definition --<br/>Identifiable collection of spatial data.<br/><br/>-- Description --<br/>NOTE The type SpatialDataSet is offered as a pre-defined type for spatial data sets. The scope of this type is packaging pre-defined data sets for the non-direct access download service. It may be reused in thematic application schemas, but if it does not fit an application-schema-specific data set object should be modelled. This type may have the same name and definition like this type, but may have a different set of properties.<br/><br/>This type specifies three properties: an external object identifier, a container for metadata (may be void), and an association to zero or more spatial objects.<br/>
