-- Name --<br/>GridObservation<br/><br/>-- Definition --<br/>Observation representing a gridded field at a single time instant.<br/>-- Description --<br/>A GridObservation is an observation of some phenomenon (or phenomena) over a gridded field. E.g. Output from a model, or rectified, georeferenced satellite data.<br/>The result of a GridObservation is a discrete coverage within a compound spatiotemporal CRS where the domain consists of a two- or three-dimensional grid of points, all having the same time instant temporal component.<br/>
