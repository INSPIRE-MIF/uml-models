Represents a location relative to a network.<br/><br/>The <i>reference</i> is the label which is assigned to a <i>location</i>. With a single LRM, one <i>reference</i> must define unambiguously and exactly one <i>location</i> in the location referencing system. The<i> reference</i> is the string of data which is passed between different implementations of a location referencing system to identify the <i>location</i>.<br/>[ISO 17572-1]<br/>
